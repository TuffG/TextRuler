# Resources Directory

This directory contains application resources such as icons and images.

## Required Files

- `icon.png` - System tray icon (optional, will use default Qt icon if not found)

## Icon Requirements

- Format: PNG
- Recommended size: 16x16, 32x32, or 64x64 pixels
- Transparent background recommended

If the icon file is not present, the application will automatically use a default Qt icon.
